# Fundamentos Algorítmicos 2

Este repositorio contiene la estructura central y material evaluativo del curso de **Fundamentos Algorítmicos 2**. Está diseñado bajo estrictas pautas corporativas para introducir a futuros analistas en las metodologías y arquitecturas de alto calibre utilizadas en el desarrollo de software.

## Estructura del Conocimiento

La información de estudio y auditoría se despliega secuencialmente a través de 3 módulos (carpetas), cada uno enfocado en potenciar la lógica temporal/espacial mediante pseudocódigo, sin atascos de lenguaje.

*   📂 **Clase_2: Análisis de Complejidad y Búsqueda Avanzada**
    *   Teoría asintótica (Big O, Omega, Theta). Diferencias de peso logarítmico entre Búsqueda Lineal y Binaria.
    *   Fundamentos lógicos en arquitecturas de memoria como Listas Enlazadas, Pilas (Stacks/LIFO) y Colas (Queues/FIFO).
*   📂 **Clase_3: Ordenamiento Avanzado y Grafos**
    *   Sustitución de modelos ineficientes (Bubble Sort) por arquitecturas "Divide y Conquista" imperativas como **Merge Sort** y **Quick Sort**.
    *   Topologías de exploración: Redes profundas limitantes (DFS) contra rastreadores concéntricos por niveles (BFS). 
*   📂 **Clase_4: Programación Dinámica (DP) y Optimización**
    *   Dominio del manejo caché global (Top-Down/Memorización y Bottom-Up/Tabulación) para aislar procesos algorítmicos redundantes.
    *   Auditorías y simulación de fracaso temporal evaluando DP contra el método *Greedy (Algoritmo Avaro)* en el mundo real.

## Dinámica Integradora Estudiantil

Dentro de cada módulo ubicarás estrictamente dos documentos:
1.  📄 **`Contenido.md`**: Es la unidad de estudio. Alberga fundamentaciones conceptuales, deducciones, casos asintóticos y base teórica pura descrita con ejemplos corporativos o lógicos diarios para su fácil digestión.
2.  📄 **`Taller.md`**: Material transaccional y evaluable aplicable directamente a consola. Aquí el estudiante validará frente al área analítica sus apropiaciones técnicas respondiendo lógicamente, generando código (pseudocódigo) y desarrollando evaluaciones asíncronas estructuradas.

## Herramientas del Entorno Local

Para la asimilación del Taller, se exige la destreza sobre utilidades de análisis industrial distribuidas a lo largo de las clases:
*   [PSeInt](http://pseint.sourceforge.net/): Compilador pseudológico base para estructuras funcionales de Pila y bucles genéricos.
*   [Flowgorithm](http://www.flowgorithm.org/download/): Audita la lógica del estudiante limitando de forma visual flujos continuos interrelacionados.
*   [Thonny](https://thonny.org/): Ideal para aplicar validaciones de vector por modo Debug paso a paso introduciéndote al ambiente de código real.
*   [Replit](https://replit.com/): Consolidación asíncrona online del aprendizaje para codificación en pares.

> **Aviso para Analistas / Estudiantes:**  El contenido de este dominio no asume ligerezas en sus instrucciones. Sigue las lógicas planteadas con rigor investigativo.
